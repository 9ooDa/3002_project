This is a Readme file that explains the files and codes in this zip folder.

The "final_code.ipynb" file contains the codes used for the data pre-processing and the accuracies measurements for 5 different machine learning models: KNN, Decision Tree, Gaussian NB, Logistic Regression, and LDA.
This file also includes the codes used for the segmentation value prediction of our test dataset and the the clustering method using PCA.

The "K-means.py" file contains the codes used for implementing K-means clustering method on the training and testing dataset.

The "Train.csv" and "Test.csv" files contain our training and testing datasets, and the "Prediction.csv" file is the resulted csv file that includes the predicted segmentation group values for each customers in testing dataset.
